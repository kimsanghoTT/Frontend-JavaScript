변수(Variable)

        * 메모리에 값을 계속 바꿔가면서 저장할 수 있는 공간 - 변하는 수
        * 변수에 이름을 붙이고(변수명) 원하는 값을 넣어주면
          변수명이 필요한 곳에서 변수명을 호출해서 지정된 값을 사용할 수 있음

          
변수명 이름 작명 규칙

          - 의미있는 이름 작성.(a, b, c 식으로 작성 지양)
          - 대소문자 구분
          - 특수문자는 _(언더스코어)만 사용 가능 (단 첫 글자는 제외)
          - 중간에 대문자를 작성해주는 카멜표기법으로 작성

        [카멜 표기법(Camel Case)]
        - 여러 단어를 변수명으로 작성할 때 이어지는 첫 번째 단어를 대문자로 작성
        ex) memberName

        변수명 명명 규칙
        특수 문자 중에서 "$" :
        단독으로 사용하거나, 숫자랑 사용하거나, $변수명 식으로 맨 앞에 사용 가능
            ->추후 JQuery 표기 문법으로 사용할 문자

        $ 문자는 어떤 표현식을 표현할 때 쓰기도 함

Java Script 변수의 종류와 특징

        const : 상수, 변수명 중복 X, 가장 많이 사용
            -> 변수명 재사용 불가능, 변수 안에 있는 내용 변경 불가능
        var : 변수, 변수명 중복 O (덮어쓰기 가능)
            -> 변수명 재사용 가능, 변수 안에 있는 내용 변경 가능
        let : 변수, 변수명 중복 X
            -> 변수명 재사용 불가능, 변수 안에 있는 내용 변경 가능

자바스크립트의 자료형

        - 문자열을 String 대신에 "" 표현 후 작성
        const 문자열 = "hihi";

        - 정수를 표현할 때는 int 숫자 작성
        const 정수 = 1;

        - 논리값(boolean)을 표현할 때는 true, false 작성
        const 논리 = true; ->진실

        - null -> 참조하는 값이 없음 *주로 오타에서 발생
