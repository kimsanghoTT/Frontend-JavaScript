배열 Array

    - 변수를 묶음으로 다루는 것

    -> 자료형 상관없이 여러 값을 한 번에 다룸

    -JS 배열 특징
        1. 자료형 제한이 없음.(String, int 구분 x, 보통은 한 가지 자료형만 저장)
        2. 길이 제한 x (자동으로 길이 조절함)

    [배열 선언 방법]
    1. const arr = new Array(); => 0칸 짜리 비어있는 배열 생성
    2. const arr = new Array(3); => 값이 비어있는 3칸 짜리 배열 생성
    3. const arr = [];   => 0칸짜리 비어있는 배열 생성
    4. const arr = [값1, 값2, 값3]; => 3칸짜리 배열 생성, 각 칸의 값이 채워져있음
