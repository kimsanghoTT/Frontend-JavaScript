attr

        attr(attribute) 속성
        jQuery에서 HTML 태그 속성을 가져오거나 설정할 때 사용되는 기능

        jQuery를 사용하지 않을 때 속성 사용방법
        getAttribute = 속성 정보 가져오기
        getAttribute('원하는속성선택');

        setAttribute = 속성 정보 변경하기
        setAttribute('원하는속성선택', '변경할값작성');

        jQuery를 사용했을 때 속성 사용방법
        attr만 작성하면 구분x

        attr('원하는속성선택');
        attr('원하는속성선택', '변경할값작성');

        (ex)
        
        src, alt, style, class -> 모두 attr(속성)에 해당함
        즉, attr은 태그 안에 작성된 속성값을 가져오거나 변경, 제거할 때 사용

        1. 태그 안에 있는 속성값 가져오기
            $("img").attr('src'); -> src 안에 있는 그림주소값을 가져옴

        2. 태그 안에 있는 속성값 변경하기
            $("img").attr('src', '변경하고싶은 주소값'); 
                -> src안에 기존에 있던 주소를 다른 주소로 변경

        3. 태그 안에 있는 속성값을 여러개 변경하기
            $("img").attr({src : "변경할주소값", alt : "변경할그림설명"});
