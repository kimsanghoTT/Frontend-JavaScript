자바스크립트 언어를 간편하게 사용할 수 있도록 단순화시킨 
        오픈소스 기반의 자바스크립트 라이브러리

        사용 방법
        1. 다운로드 : jQuery 홈페이지에서 다운로드
            제이쿼리 다운로드하기
            
            

        2. CDN(Content Delivery Network) : 주소 참조 활용
            
            

        [기본문법]
        1. id문법  
        - $(document).ready(function(){ -> 문서에서 어떠한 행동을 할 준비가 됨
            $("#id이름").click(function(){ -> id이름을 클릭했을 때 할 동작
                동작에 관련된 코드 작성
            })
        })

        2. 이벤트리스너 문법 (onclick, keyon, onchange 등)
        function 이벤트리스너에작성한이름(){
            $("#아이디명").기능작성();
        }

        [선택자문법]
        1. css 선택 사용
            $("태그명 or #id명 or .class명").css("서식");

        2. toggle 선택 사용
            $("태그명 or #id명 or .class명").toggle("속도조절"); -> 속도 생략 가능

        3. slideToggle 선택 사용
            토글과 비슷하지만 주로 이미지를 보이거나 숨길 때 사용

        4. hide() show() 선택 사용
            숨기기와 보이기

        5. append() 선택 사용
            태그나 값을 추가할 때 사용
